Spellbound Rollers

Tags:

  • Unity
  • Game Jam

Release Date: 17 July 2022

Description

A simple survival game where you have to defend yourself against living dice with your own through utilizing several special powers: Fire, poison, ice etc... Collect the chips, gain XP and unlock more powers!

Features that I did:

  • Character Movement script (designer and artist changed the movement parameters)
  • Dialogue System (Designer gave the script in a design document and I implemented it. Sidenote: Trying to convert the editor UI to UI toolkit to learn UI Toolkit. Link
